# 📊 Walmart Sales Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Sales Analytics Dashboard built using Microsoft Power BI based on a real-world Walmart Sales dataset.

This is my first project working with a real dataset, where I focused on transforming raw data into meaningful insights through data cleaning, modeling, and visualization.

---

## 📂 Dataset

The dataset used in this project is the Walmart Sales Dataset sourced from Kaggle.

📎 Dataset link is provided in the `dataset-link.txt` file.

---

## 🛠 Tools & Technologies

* Microsoft Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation

## 📊 Dashboard Features

### 🔹 Key Performance Indicators (KPIs)

* **Total Sales:** 6.74 Billion
* **Average Sales:** 1.05 Million
* **Holiday Sales:** 505.30 Million
* **Non-Holiday Sales:** 6.23 Billion

---

### 🔹 Monthly Sales Trend (Line Chart)

* Displays weekly sales trends across the year
* Highlights seasonal patterns:

  * Growth from January to March
  * Decline in April, June, and November
  * Recovery in December

---

### 🔹 Total Sales by Store (Bar Chart)

* Compares performance across 45 stores
* Identifies top-performing and low-performing stores

---

### 🔹 Sales vs Temperature (Scatter Plot)

* Analyzes relationship between temperature and sales
* **Observation:** No strong correlation between temperature and overall sales

---

### 🔹 Interactive Slicer

* Date hierarchy filter (Year → Month → Day)
* Enables dynamic and interactive analysis

---

## 🔍 Key Insights

* Sales show clear seasonal trends with fluctuations across months
* Some stores consistently outperform others
* External factors like temperature have minimal impact on overall sales
* Holiday sales contribute significantly but are smaller compared to non-holiday sales
